# jamstackfns

## Running Locally

```bash
$ git clone https://github.com/leerob/jamstackfns.git
$ cd jamstackfns
$ yarn
$ yarn dev
```

To use anything that requires API routes, you will need to run `now dev` and have a `.env` file similar to this.

```
BUTTONDOWN_API_KEY=
TWITTER_CONSUMER_KEY=
TWITTER_CONSUMER_SECRET=
SESSION_SECRET=
SENDGRID_API_KEY=
```
