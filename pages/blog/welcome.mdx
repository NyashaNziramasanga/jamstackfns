[JAMstack](https://jamstack.org/) (JavaScript, APIs, and Markup) is fundamentally reshaping the web.
Coined by [Netlify's co-founder](https://en.wikipedia.org/wiki/Netlify#JAMstack), JAMstack bridges
the gap between static and dynamic websites. Capitalize on [static site performance](https://leerob.io/blog/next-prisma),
security, and scalability while still having dynamic data.

JAMstack Functions is a directory of the best serverless functions for JAMstack
applications.
